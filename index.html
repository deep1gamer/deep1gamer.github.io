MineServ

Run Minecraft servers easily on PC and Android with MineServ.

⬇ Download for Windows (.exe) ⬇ Download for Android (.apk)

About the App

The app allows you to create and manage Minecraft servers for free, although it's currently available only for Windows. A mobile version is under testing.

It requires a good PC to handle a Minecraft server, since it runs locally. We recommend at least 8–12 GB of RAM for a 4 GB server, along with a good CPU and GPU.

For more details, you can visit the GitHub repository.

How to Use the App

  • First install the server using the install button and wait until the terminal says "installed".
  • Then click the start button to launch the server.
  • The app has three tabs: Server Output, Plugins, Browser, and a Settings tab (currently inactive).

Features

  • Create and manage Minecraft servers easily
  • Add and manage plugins
  • Web version available soon (runs on Google Colab, not your PC)
